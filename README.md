💾 Gestión de tarjeta SD (ESP32)

Este proyecto permite trabajar con una tarjeta microSD conectada a una ESP32 para almacenar y gestionar datos.

🔧 Funcionalidad
Crear y borrar archivos
Escribir y leer datos
Crear y eliminar directorios
Listar contenido y mostrar estructura

Ejemplo:

/practica
  ├── info.txt
  ├── logs/
  └── datos/
🔍 Notas
La tarjeta debe estar en FAT32
Comunicación mediante SPI
Requiere módulo microSD
🚀 Uso
Conectar módulo SD
Subir código
Abrir monitor serie
Ejecutar operaciones automáticamente
